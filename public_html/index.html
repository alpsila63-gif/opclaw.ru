Вопросы

Частые вопросы

Чем отличаются OpenClaw, Claude Code и Codex?
+
OpenClaw — open-source AI-агент для мессенджеров, автоматизации и голосового управления (15+ платформ). Claude Code — агентный инструмент от Anthropic для программирования из терминала (agentic coding). OpenAI Codex — облачный AI-агент для разработки с мультиагентными workflow. Все три можно внедрить в комплексе или по отдельности под ваши задачи.
Что нужно для начала работы?
+
Для OpenClaw: VPS-сервер или локальный ПК с Node.js и API-ключ от AI-модели (Claude, GPT, DeepSeek). Для Claude Code: подписка Anthropic Pro/Max. Для Codex: подписка OpenAI. Всё остальное — установку, настройку и интеграцию — я сделаю за вас.
Сколько стоит использование AI-моделей?
+
OpenClaw бесплатен — вы платите только за API-ключи: Claude Sonnet ~$3 за 1 млн токенов, GPT-5 Codex ~$2.50. Claude Code Pro — $20/мес, Max — $100/мес. OpenAI Codex доступен от $20/мес. Для типичного использования OpenClaw это $5–20 в месяц.
Можно ли запустить без VPS, на обычном ПК?
+
Да, OpenClaw отлично работает локально на Windows, macOS и Linux. Но для круглосуточной работы рекомендую VPS — от 500 руб./мес.
Безопасно ли это?
+
OpenClaw работает локально — ваши данные не уходят на чужие серверы. Однако важно правильно настроить безопасность: ограничить доступ, использовать надёжные пароли, не ставить непроверенные Skills. Я помогу с настройкой безопасности.
Почему так недорого по сравнению с агентствами?
+
Я работаю как фрилансер без офиса и штата менеджеров. Нет накладных расходов — только честная оплата за работу. Агентства берут 200–800 тыс. руб. за аналогичные задачи.
Какие мессенджеры поддерживаются?
+
WhatsApp, Telegram, Slack, Discord, Signal, Google Chat, Microsoft Teams, Matrix, iMessage (через BlueBubbles), Zalo, WebChat и другие. Полный список — более 15 платформ.
